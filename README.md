# CollabRift AI Workspace

AI-powered real-time collaboration workspace built with React, TypeScript, Redux Toolkit, and Vite.

## Tech Stack

- React 19
- TypeScript
- Vite
- Redux Toolkit
- Tailwind CSS
- Framer Motion

## Features

- Real-time collaboration
- AI assistant
- Workspace management
- Team communication
- Task tracking
- Responsive UI

## Getting Started

```bash
npm install
npm run dev
